The workflow to generate the index (ToC) for a file foo.tex is:
\begin{itemize}
\item \verb|xelatex foo|
\item \verb|splitindex -- -s plain.ist -c|
\item \verb|xelatex foo|
\end{itemize}

where the index style file (.ist) is:

\begin{verbatim}
delim_0 "\\space\\dotfill\\space "\hss
delim_1 "\\space\\dotfill\\space "\hss
delim_2 "\\space\\dotfill\\space "\hss
delim_n ", "
delim_r "--"
delim_t ""
encap_prefix ""
encap_suffix ""
\end{verbatim}
